# Synnovator UI 设计规范

本文档定义 Synnovator Hackathon 平台的完整 UI 设计规范，基于 Neon Forge 主题和 16 个用户旅程。

**设计系统：** Neon Forge **前端框架：** Next.js 14 \+ Tailwind CSS v4 \+ shadcn/ui **参考平台：** Devpost, HackerEarth, MLH, Devfolio

---

## shadcn/ui 组件映射表

以下表格列出 UI 设计中所有组件与 shadcn/ui 组件的对应关系，用于快速原型实现。

### 布局组件

| UI 组件 | shadcn 组件 | 备注 |
| :---- | :---- | :---- |
| Header | 自定义布局 | 使用 flex 布局 |
| Sidebar | `sidebar` | 使用 sidebar-07 或 sidebar-10 变体 |
| Panel | `sheet` / `card` | 右侧面板用 sheet，内容区用 card |
| Body | 自定义布局 | 使用 flex-grow \+ overflow-auto |
| Footer | 自定义布局 | 使用 flex 布局 |
