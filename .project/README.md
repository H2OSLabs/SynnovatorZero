# Project Management

This directory contains project management files used for planning and tracking work.

## Files

- `task_plan.md` — High-level task tracking with phases and status
- `progress.md` — Detailed session-by-session progress log
- `findings.md` — Key discoveries and decisions made during development

## Usage

These files are used by the `planning-with-files` skill to maintain context across sessions.
See `.claude/skills/planning-with-files/SKILL.md` for more details.
