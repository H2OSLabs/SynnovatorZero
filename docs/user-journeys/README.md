# 用户旅程

本文档描述 Synnovator 平台的完整用户旅程，面向开发团队，涵盖从内容浏览到活动参与、团队协作、内容创作、社交互动等全流程场景。

> 角色定义详见 [data-types.md](../data-types.md#角色定义)，数据类型详见 [data-types.md](../data-types.md)，关系详见 [relationships.md](../relationships.md)。
