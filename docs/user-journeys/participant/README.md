## 1. 内容浏览与发现

**核心目标：** 快速发现感兴趣的内容（提案、帖子、活动）并建立初步连接。
详见：[01-browsing.md](./01-browsing.md)

- **首页 Feed 流**: 聚合全站优质内容，支持 Proposal/Team/General 并行分类。
- **全局导航**: 探索、营地、星球、资产四大入口。
- **搜索与筛选**: 全局关键词检索与 Tag 过滤。

## 2. 社交互动

**核心目标：** 与他人建立联系，表达态度，形成社区网络。
详见：[05-social.md](./05-social.md), [06-networking.md](./06-networking.md)

- **互动行为**: 点赞、收藏、评论、分享。
- **关系网络**: 关注 (Follow) 与 好友 (Friend) 体系。
- **通知中心**: 实时接收互动反馈与系统通知。

## 3. 活动参与

**核心目标：** 报名参加活动，提交作品，获取奖励。
详见：[02-participation.md](./02-participation.md), [03-team-collaboration.md](./03-team-collaboration.md)

- **报名流程**: 浏览 -> 报名 (Rule Check) -> 组队/个人 -> 提交提案。
- **团队协作**: 寻找队友、组建团队、团队内部资产共享。

## 4. 内容创作

**核心目标：** 发布高质量内容，管理版本，沉淀资产。
详见：[04-creation.md](./04-creation.md)

- **创作类型**: 帖子 (Post)、提案 (Proposal)、资源 (Resource)。
- **版本管理**: 草稿、发布、迭代快照。

## 5. 个人中心与资产

**核心目标：** 展示自我形象，管理数字资产和荣誉。
详见：[07-profile.md](./07-profile.md), [09-asset-management.md](./09-asset-management.md)

- **个人主页**: 展示创作、社交关系与荣誉。
- **资产库**: 统一管理文件、证书、勋章及算力配额。
- **营地 (Camp)**: 个人的内容管理中心。
