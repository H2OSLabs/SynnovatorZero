# 16. 资产管理 (Asset Management)

- **角色：** 已登录用户
- **前置条件：** 已登录

用户通过左侧导航栏的“资产”入口进入，集中管理个人持有的各类数字资产，包括创作内容、上传文件、荣誉证明及算力资源。

## 16.1 资产概览 (Asset Dashboard)

资产页面将用户的资源分为以下四大类进行展示和管理：

| 资产类型 | 说明 | 包含内容 | 数据操作 |
|---------|------|---------|---------|
| **我的文件** | 平台内原生的创作内容 | Markdown 文档、笔记、代码片段 | `READ resource` (type=document) |
| **上传文件** | 外部导入的静态资源 | 图片、PDF、数据集、压缩包 | `READ resource` (type=file/image) |
| **荣誉证明** | 参与活动获得的认证与奖励 | 数字证书 (Certificates)、成就勋章 (Badges) | `READ user_achievements` |
| **算力资源** | 平台分配的计算能力 | CPU/GPU 时长、存储空间配额 | `READ user_compute_quota` |

## 16.2 文件与资源管理

管理用户主动创建或上传的非结构化数据。

| 用户旅程 | 说明 | 数据操作 |
|---------|------|---------|
| **统一视图** | 在一个列表中查看所有文件，支持按类型（文档/图片/附件）切换视图 | `LIST resource` (filter by type) |
| **资产检索** | 通过文件名、标签或上传时间搜索特定资产 | `SEARCH resource` |
| **关联查询** | 查看某个资产当前被哪些帖子、提案或活动引用（引用计数） | `READ resource:references` |
| **资产清理** | 删除不再使用的资产；若资产被引用，系统应提示风险或禁止删除 | `DELETE resource` (check ref_count) |
| **下载/导出** | 将云端资产下载到本地设备 | `DOWNLOAD resource` |

## 16.3 荣誉体系 (Achievements)

展示用户在平台活动中的成就记录。

| 用户旅程 | 说明 | 数据操作 |
|---------|------|---------|
| **证书管理** | 查看活动完赛、获奖等电子证书，支持高清预览和下载 PDF | `READ certificate` |
| **勋章佩戴** | 查看所有解锁的勋章，并选择特定勋章在个人主页名字旁展示 | `UPDATE user:profile` (equipped_badge) |
| **荣誉分享** | 生成荣誉分享卡片，用于社交媒体传播 | `GENERATE share_card` |

## 16.4 算力资源 (Computing Resources)

针对开发者和科研用户的计算资源管理面板。

| 用户旅程 | 说明 | 数据操作 |
|---------|------|---------|
| **额度监控** | 实时显示剩余算力点数、GPU 时长及存储空间使用率 | `READ compute_stats` |
| **消耗明细** | 查看算力消耗流水，关联到具体的 Notebook 任务或活动项目 | `READ compute_logs` |
| **资源申请** | 针对高算力需求的活动，提交额外的资源扩容申请 | `CREATE resource_request` |

## 16.5 异常与配额限制 (Constraints & Quotas)

| 场景 | 规则说明 | 系统行为 |
|------|---------|---------|
| **存储配额超限** | 用户上传文件总大小超过 `storage_quota` | **禁止新文件上传**，提示清理旧文件或申请扩容；但不影响已有文件的查看和下载 |
| **算力耗尽** | 用户的算力点数 (`compute_points`) 归零 | **强制停止**正在运行的 Notebook/训练任务；禁止启动新任务 |
| **文件类型限制** | 上传禁止的文件类型（如 .exe, .sh 等高危脚本） | 系统直接拒绝上传，返回 `400 Bad Request` |
| **资产归属冲突** | 尝试删除被他人引用的共享资产 | 若引用计数 `ref_count > 0`，系统警告或**禁止删除**（取决于系统配置），建议使用“归档”而非物理删除 |
