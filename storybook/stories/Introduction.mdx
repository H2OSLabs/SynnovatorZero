## Components

This Storybook includes the following UI components:

- **Button** - 6 variants, 8 sizes, icon support
- **Badge** - 6 variants with icon support
- **Card** - Composable card with header, content, footer, action
- **Tabs** - Default and line variants, horizontal/vertical orientation
- **Input** - Text input with focus and validation states
- **Avatar** - Multiple sizes, badges, groups
- **Tooltip** - Positioned tooltips with arrow
- **Separator** - Horizontal and vertical dividers
- **ScrollArea** - Custom scrollbar areas
- **DropdownMenu** - Full menu with checkbox, radio, submenu support

Each component has a **Neon Forge Style** story showing platform-specific usage.
