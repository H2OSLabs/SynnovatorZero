# Neon Forge / 霓虹锻造

A vibrant, high-energy dark theme with electric neon accents, designed for the Synnovator (协创者) creative collaboration platform. Bold lime green leads a spectrum of vivid hues against deep charcoal backgrounds.

高能量暗色主题，以明亮霓虹色为点缀，专为协创者 (Synnovator) 创意协作平台设计。以亮绿色为主色调，搭配多彩荧光色系与深色背景。

## Color Palette / 色彩体系

### Brand Colors / 品牌色
- **Lime Green / 亮绿 (Primary / 主色)**: `#BBFD3B` - Core brand accent, CTAs, highlights / 核心品牌色，行动按钮，高亮
- **Mint / 薄荷绿**: `#74FFBB` - Secondary accent, success states, tags / 辅助色，成功状态，标签
- **Cyan / 青色**: `#41FAF4` - Tertiary accent, links, interactive elements / 第三色，链接，交互元素
- **Pink / 粉色**: `#FF74A7` - Feature cards, alerts, decorative accents / 功能卡片，提醒，装饰
- **Blue / 蓝色**: `#4C78FF` - Info states, official badges, navigation highlights / 信息状态，官方徽章，导航高亮
- **Orange / 橙色**: `#FB7A38` - Hot/trending indicators, energy accents / 热门/趋势指示，能量色

### Neutral Colors / 中性色
- **Dark BG / 深色背景**: `#222222` - Primary background / 主要背景
- **Secondary BG / 次级背景**: `#333333` - Cards, panels, elevated surfaces / 卡片，面板，悬浮层
- **Surface / 底层**: `#181818` - Deepest background layer / 最深背景层
- **White / 白色**: `#FFFFFF` - Primary text on dark backgrounds / 深色背景上的主要文字
- **Near Black / 近黑**: `#00000E` - Text on light/accent backgrounds / 亮色/强调色背景上的文字
- **Muted / 灰色**: `#8E8E8E` - Secondary text, placeholders, captions / 次要文字，占位符，说明文字
- **Light Gray / 浅灰**: `#DCDCDC` - Subtle text, disabled states / 弱化文字，禁用状态

### Semantic Colors / 语义色
- **Success / 成功**: `#00B42A` - Positive feedback / 正向反馈
- **Error / 错误**: `#FA541C` - Error states, destructive actions / 错误状态，危险操作
- **Warning / 警告**: `#FAAD14` - Caution indicators / 注意提示

### Accent Colors / 点缀色
- **Light Green / 亮绿**: `#8AFF80`
- **Yellow / 亮黄**: `#F0FF31`
- **Purple / 紫色**: `#E3B8FF`
- **Rose / 玫红**: `#FBC2DA`

## Typography / 字体体系

### Font Families / 字体族
- **Headers / 标题**: OPPOSans Bold (700)
- **Body Text / 正文**: OPPOSans Regular (400)
- **Secondary Text / 辅助文字**: Source Han Sans SC Regular (思源黑体)
- **Numbers & Code / 数字与代码**: Poppins Medium (500)

### Type Scale / 字号体系
- **24px** - Heading / 标题文字
- **20px** - Subheading / 副标题
- **18px** - Section Title / 区块标题
- **16px** - Body Large / 正文大字
- **14px** - Body / 正文
- **12px** - Caption / 说明文字
- **10px** - Overline / 小标注

### Font Weights / 字重
- **300** - Light / 细体
- **400** - Regular / 常规
- **500** - Medium / 中等
- **700** - Bold / 粗体

## Spacing Tokens / 间距体系

- **XS (4px)**: Tight spacing / 紧凑间距 — icon gaps, tag padding / 图标间距，标签内距
- **SM (8px)**: Default inner padding / 默认内距
- **MD (16px)**: Standard gap between elements / 元素间标准间距
- **LG (24px)**: Section gaps / 区块间距
- **XL (32px)**: Major section padding / 大区块内距

## Border Radius Tokens / 圆角体系

- **SM (4px)**: Tags, small badges / 标签，小徽章
- **MD (8px)**: Inputs, small cards / 输入框，小卡片
- **LG (12px)**: Cards, panels, color swatches / 卡片，面板，色卡
- **XL (21px)**: Buttons, pill-shaped tabs / 按钮，胶囊标签
- **Pill (50px)**: Circular avatars, rounded pills / 圆形头像，全圆角药丸

## Best Used For / 适用场景

Creative collaboration platforms, startup pitch decks, innovation hubs, team-building tools, social creation apps, hackathon platforms, neon-on-dark UI designs, Gen-Z/millennial-facing products.

创意协作平台、初创企业路演、创新中心、团队协作工具、社交创作应用、黑客松平台、霓虹暗色 UI 设计、面向 Z 世代/千禧一代的产品。
